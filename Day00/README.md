# Subject: Day 00 - Java Fundamentals

## Objectives
Java dilinin temel sözdizimini, C++'tan ayrılan bellek yönetim mantığını ve standart çıktı mekanizmalarını kavramak.

## Instructions
- **Ex00 & Ex01:** Ekrana veri yazdırma (`System.out.println`) ve temel veri tipleriyle (int, double vb.) basit matematiksel işlemler yapın.
- **Ex02 (Pass-by-Value):** Bir metodun içine hem ilkel (primitive) bir değişken hem de bir nesne göndererek Java'nın "Değerle Çağırma" (Pass-by-Value) mantığını kanıtlayın. Nesnenin referansının kopyalandığını ve içeride yapılan değişikliğin dışarıyı nasıl etkilediğini gözlemleyin.
