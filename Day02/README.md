# Subject: Day 02 - Inheritance & Hierarchy

## Objectives
Sınıflar arası hiyerarşi kurarak kod tekrarını önlemek ve `extends` anahtar kelimesinin kullanımını öğrenmek.

## Instructions
- Mevcut `Warrior` sınıfını bir "Üst Sınıf" (Superclass) olarak belirleyin.
- `Warrior` sınıfından türeyen `Mage` ve `Archer` gibi "Alt Sınıflar" (Subclass) oluşturun.
- Alt sınıfların constructor yapısında `super()` anahtar kelimesini kullanarak üst sınıfın constructor'ını doğru şekilde tetikleyin.
- Ortak özelliklerin üst sınıfta, sınıfa özel yeteneklerin ise alt sınıflarda toplandığından emin olun.
